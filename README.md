# Vehicle-History-Tracker
# 🚗 Vehicle History Tracker (Soroban Smart Contract)

## 📌 Project Description
Vehicle History Tracker is a decentralized application built using the Stellar Soroban smart contract platform. It enables secure and transparent storage of a vehicle’s history using a unique Vehicle Identification Number (VIN).

This system ensures that vehicle records such as ownership, maintenance, and incidents are immutable and easily accessible, helping build trust in vehicle transactions.

---

## ⚙️ What it does

- Registers vehicles using a unique VIN
- Stores vehicle owner information
- Records vehicle history events (repairs, accidents, servicing, etc.)
- Retrieves complete vehicle data anytime from the blockchain

All operations are handled through a Soroban smart contract, ensuring decentralized and tamper-proof data storage.

---

## ✨ Features

- 🔐 **Decentralized & Secure**  
  Data is stored on-chain using Stellar Soroban.

- 🚘 **Vehicle Registration**  
  Add new vehicles with a unique VIN.

- 🧾 **History Tracking**  
  Maintain logs such as:
  - Repairs
  - Accidents
  - Maintenance
  - Ownership updates

- 🔍 **Easy Data Retrieval**  
  Fetch full vehicle details or only history records.

- ⚡ **Efficient Smart Contract**  
  Lightweight and optimized for low storage cost.

---

## 🧠 Smart Contract Functions

### 1. `register_vehicle(vin, owner)`
Registers a new vehicle on the blockchain.

### 2. `add_record(vin, record)`
Adds a new history entry to the vehicle.

### 3. `get_vehicle(vin)`
Returns complete vehicle details.

### 4. `get_vehicle_history(vin)`
Returns only the history records.
